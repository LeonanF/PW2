Estrutura do Curso

1º Semestre

  • Introdução à Ciência de Dados
  • O ciclo de vida de um projeto de Ciência de Dados
  • Exploração de dados e pré-processamento
  • Noções básicas de programação em Python ou R
  • Ambientes de desenvolvimento e ferramentas para Ciência de Dados

2º Semestre

  • Conceitos fundamentais de estatística
  • Distribuições de probabilidade e testes estatísticos
  • Análise exploratória de dados
  • Visualização de dados
  • Amostragem e estimação

3º Semestre

  • Introdução ao Aprendizado de Máquina
  • Tipos de aprendizado: supervisionado, não supervisionado e por reforço
  • Algoritmos de classificação e regressão
  • Algoritmos de agrupamento (clusterização)
  • Avaliação de modelos de Machine Learning

4º Semestre

  • Introdução ao Big Data e sua importância para a Ciência de Dados
  • Sistemas de armazenamento e processamento distribuídos (Hadoop, Spark)
  • Trabalhando com dados em larga escala
  • Análise de streaming de dados

5º Semestre

  • Conceitos de mineração de dados e suas aplicações
  • Pré-processamento de dados para mineração
  • Algoritmos de mineração de dados.
  • Mineração de texto e processamento de linguagem natural

6º Semestre

  • Técnicas avançadas de visualização de dados
  • Storytelling com dados
  • Design de visualização efetiva
  • Comunicação de resultados e relatórios de Ciência de Dados

7º Semestre

  • Aprendizado de Máquina avançada.
  • Processamento de imagem e visão computacional
  • Processamento de áudio e reconhecimento de fala
  • Análise de dados geoespaciais
  • Aplicações éticas e responsáveis de Ciência de Dados

8º Semestre

  • Desenvolvimento de um projeto prático de Ciência de Dados
  • Coleta, limpeza e análise dos dados
  • Modelagem, treinamento e avaliação de um modelo de Machine Learning
  • Visualização e interpretação dos resultados
  • Apresentação final do projeto
