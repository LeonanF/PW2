Curso de ciência de dados:

A Ciência de Dados é uma área interdisciplinar que utiliza métodos e técnicas de Estatística, Matemática e programação para extrair conhecimento de grandes conjuntos de dados.

Um dado é o menor nível de abstração de uma informação, uma forma bruta de conhecimento que ainda não foi devidamente tratada para, de fato, fornecer algum insight. Ele pode ser dos mais diferentes formatos, como textos, números, imagens, áudio ou vídeo.

O papel do cientista de dados surgiu na academia: as universidades necessitavam de programadores capazes de trabalhar em equipe. Dessa forma, foram criados programas como o Institute for Advanced Analytics da North Carolina State University, que ajudaram a produzir toda uma geração de cientistas de dados.
